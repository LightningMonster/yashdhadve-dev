About PlantsWorld

A modern, responsive website for a premium indoor plant store. This project showcases a beautiful collection of indoor plants with a focus on user experience and aesthetic design. The UI is designed to provide an immersive shopping experience while maintaining simplicity and elegance.

Technical Implementation

The design was crafted in Figma with a focus on modern UI/UX principles. Key technical aspects include:

  • CSS Grid and Flexbox for responsive layouts
  • Modern CSS features for smooth animations
  • Optimized assets for fast loading
  • Mobile-first design approach
  • Accessible color contrasts
